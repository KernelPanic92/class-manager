# Class Manager

## Description

Simple backend implementation of the database used for the plansoft angular course for high schools


framework used: [Nestjs](https://github.com/nestjs/nest).

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
